# 프로젝트: 편의점 최적입지 스코어링 및 검증 (서울시 중랑구 기준)

## 개요
- 목표: 서울 전체 상권·유동인구 공공데이터로 편의점 매출 결정 요인을 학습하고,
  중랑구를 50m 격자 단위로 스코어링하여 출점 후보지를 도출·검증한다.
- 검증 프레임: 점수×기존 점포 유무 4분면 분석 + 면목2동 동부시장 인근 현장 임장.
- 배경: 사용자의 졸업작품(가중치를 선행연구로 부여한 입지선정)의 한계를
  "데이터로 가중치를 학습"하는 방식으로 발전시킨 개인 포트폴리오.

## 파이프라인 (Phase)
1. 데이터 수집 → 2. 정제·통합 → 3. 격자 생성·피처 계산 → 4. 모델 학습(서울 전체)
→ 5. 중랑구 스코어링·4분면 검증·지도 → 6. README·패키징

## 작업 규칙 (중요)
- 사용자는 파이썬 기초만 아는 학습자다. 코드에는 반드시 한국어 주석을 충분히 달 것.
- 한 번에 한 단계만 진행하고, 단계가 끝나면 "방금 한 일"을 3줄로 요약할 것.
- 라이브러리는 위 설치 목록 안에서만 사용. 새 라이브러리가 필요하면 이유를 먼저 설명하고 허락받을 것.
- 좌표계는 분석용 EPSG:5179, 웹지도 출력용 EPSG:4326으로 통일.
- 의미 있는 작업 단위마다 git commit을 한국어 메시지로 남길 것.
